# Project Summary
The "Talk to My Lawyer" project is a SaaS application designed to facilitate legal document generation and review. It supports three user roles: Subscribers who generate letters, Employees who assist with sales and earn commissions through coupon codes, and Admins who review and approve the generated letters. The platform leverages AI for drafting legal documents and integrates with payment processors for subscriptions.

# Project Module Description
The application comprises several functional modules:
- **User Authentication**: Manages user roles and sessions.
- **Letter Generation**: Allows subscribers to create legal letters using AI.
- **Subscription Management**: Handles user subscriptions and coupon codes.
- **Admin Review**: Provides a dashboard for admins to review, edit, and approve letters.
- **Email Notifications**: Sends emails to users regarding their letters and subscriptions.
- **PDF Generation**: Converts approved letters into downloadable PDF format.

# File Description Inventory
- `app/api/`: Contains API routes for handling requests related to user authentication, letter generation, and subscriptions.
- `app/dashboard/`: Contains components for the admin and user dashboards, including letter management and analytics.
- `components/`: Reusable UI components used throughout the application.
- `supabase/`: Contains database migrations and functions related to the Supabase backend.
- `scripts/`: Includes scripts for database setup and administration tasks.
- `styles/`: Contains global styles for the application.
- `README.md`: Documentation for the project setup and usage.

# Technology Stack
- **Frontend**: Next.js 14, TypeScript, Tailwind CSS
- **Backend**: Supabase, OpenAI GPT-4
- **Payment Processing**: Stripe
- **Email Service**: Resend
